  • ¿Qué es la especificidad?

    • Definición de Especificidad

    Se usa para determinar qué reglas del CSS prevalecerá sobre las otras. Incluso cuenta con una tabla de valores

    Ejemplo de Especificidad

    .selectorClass { color: blue }

    #selectorID { color: peru }

    En este caso prevalecerá el atributo del selector ID por encima de Class por tener una importancia mayor. Es decir, Class también se pintará de peru
